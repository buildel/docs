# AI Chat

## Used blocks
* [Text Input](/docs/blocks/text_input)
* [Chat](/docs/blocks/chat)
* [Text Output](/docs/blocks/text_output)

:::tip
Before start, make sure you understand how workflows work. We highly recommend [Create first Workflow](/docs/guides/create-first-workflow) guide
:::

## Usage

It works pretty similar to standard ChatGPT.
1. First press **Run** button in top left corner
2. Type anything you want in [Text Input](/docs/blocks/inputs-outputs/text-input)
3. In couple seconds you'll see AI's answer in [Text Output](/docs/blocks/inputs-outputs/text-output)
